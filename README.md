# Pong
Just another pong game :)

Install: http://www.monogame.net/releases/v3.6/MonoGameSetup.exe

### Controls:  
Player1 (left)  
- W = move up  
- S = move down  

Player2 (right)  
- Up = move up  
- Down = move down  
  
Escape = exit game
